# Anime shop
## Project Description
Anime Shop is a web application focused on anime-related content, where users can browse products, explore their details, and interact with them.

The application supports basic data operations such as adding, deleting, and updating information, as well as navigation between pages and interaction with the server through requests.

The goal of the project is to create a user-friendly interface and implement the core features of a modern web application, including data handling, user interaction, and content display.
